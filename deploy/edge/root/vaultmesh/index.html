What VaultMesh is

A system for continuous integrity: state continuity, signed artifacts, and adversarial checks that make claims machine-verifiable.

  • Hash-chained ledgers (append-only receipts)
  • Ed25519 signatures (detached, offline verifiable)
  • Equivocation detection (fork / replay awareness)
  • Portable evidence bundles (export / ingest / dispute packs)

What VaultMesh is not

Not a marketing dashboard. Not "trust me." Not a single point of failure. This surface is intentionally minimal so verification stays simple. 

invariants:
  - offline verification works
  - state continuity is monotonic
  - receipts are append-only
  - distrust is expected
